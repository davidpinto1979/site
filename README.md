# Porto de Encontro na Leitura

Este repositório contém os arquivos de um pequeno site estático para o projeto "Porto de Encontro na Leitura". O objetivo é divulgar clubes de leitura ativos na cidade do Porto e fornecer informação básica sobre cada um, horários de encontros e contactos.

## Estrutura do repositório

- `index.html` – página inicial com lista de clubes e calendário de encontros.
- `clubes.html`, `eventos.html`, `sobre.html`, `contactos.html` – páginas adicionais do site.
- `styles.css` – folha de estilos comum.
- Imagens de logótipo na raiz (algumas removidas recentemente).

## Como usar

1. Clone o repositório localmente.
2. Abra os ficheiros HTML num navegador ou sirva o diretório com um servidor simples (por exemplo, `python -m http.server`).
3. Faça alterações nos conteúdos conforme necessário e commit/push para o repositório remoto.
